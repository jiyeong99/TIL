# OOP(Object Oriented Programming,객체지향프로그래밍)
### - 객체의 특징
- 객체 : 대상의 정보와 동작

  - 타입(type) : 어떤 연산자(operator)와 조작(method)이 가능한가?
  - 속성(attribue) : 어떤 상태(데이터)를 가지는가?
  - 조작법(method)  어떤 행위(함수)를 할 수 있느가?

### - 객체지향 프로그래밍이란?
  - 프로그램을 여러 개의 독립된 객체들과 그 객체들 간의 상호작용으로 파악하는 프로그래밍 방법
### - 절차지향 프로그래밍
  - 데이터와 함수로 인한 변화~~그림
